2.2 Describe whether the IT tools selected were appropriate

The tools used were very appropriate in the following ways:

  • Visual Studio Code: It is a light and a very versatile editors. The extensions available are really handy. With support for hundreds of languages, VS Code helps you be instantly productive with syntax highlighting, bracket-matching, auto-indentation, box-selection, snippets, and more.
  • Codepen: CodePen is a social development environment. At its heart, it allows you to write code in the browser, and see the results of it as you build. A useful and liberating online code editor for developers of any skill, and particularly empowering for people learning to code.
  • Trello: It is an excellent way of implementing Kanban and even working on the project text based RPG it helped keeping track of the progress and issues. You can assign a particular task to someone as well which reminds them they have the responsibility.
